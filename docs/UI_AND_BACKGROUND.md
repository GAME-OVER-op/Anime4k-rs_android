# UI, сохранённые файлы и фоновая обработка

## Material-стиль

Интерфейс переведён на светлый Material 3 стиль:

- нижняя навигация;
- вкладки: Фото, Видео, Сохранённые, Настройки;
- ElevatedCard;
- Google-подобная синяя primary color;
- аккуратные поля ввода и переключатели.

## Сохранённые

Результаты теперь пишутся не во временный cache, а в:

```text
context.filesDir/results
```

Это память приложения. Во вкладке **Сохранённые** можно:

- сохранить результат в галерею;
- поделиться;
- удалить из памяти приложения.

## Настройки

Добавлены настройки приложения:

- показывать ползунки;
- foreground service + уведомление;
- запросить разрешение на уведомления.

Если ползунки выключены, остаётся только точный ручной ввод чисел.

## Уведомления и foreground service

Во время обработки запускается `ProcessingForegroundService`, который показывает постоянное уведомление со статусом.

Это нужно, чтобы Android видел активную долгую задачу и не воспринимал апскейл как обычный фоновый процесс.
